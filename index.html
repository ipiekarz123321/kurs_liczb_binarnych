Moduł 1: Jak czytać binarnie?

Pamiętaj: każda pozycja w liczbie binarnej to kolejna potęga liczby 2, idąc od prawej strony.

128 64 32 16 8 4 2 1
0 0 0 1 0 1 0 1

Powyższa liczba to: 16 + 4 + 1 = 21.

📝 Ćwiczenie 1

Przelicz liczbę binarną 00101100 na system dziesiętny.

Sprawdź odpowiedź
Mamy jedynki na pozycjach: 32, 8 oraz 4.
32 + 8 + 4 = 44

Moduł 2: Jak pisać binarnie?

Aby zamienić naszą liczbę na binarną, zadajemy sobie pytania: "Czy ta potęga dwójki się zmieści?". Jeśli tak, wpisujemy 1 i odejmujemy tę wartość.

Przykład dla liczby 13:

  • Czy 8 się mieści w 13? TAK. (Zostaje 5) -> Piszemy 1
  • Czy 4 się mieści w 5? TAK. (Zostaje 1) -> Piszemy 1
  • Czy 2 się mieści w 1? NIE. -> Piszemy 0
  • Czy 1 się mieści w 1? TAK. -> Piszemy 1

Wynik: 1101

📝 Ćwiczenie 2

Zapisz liczbę 50 w systemie binarnym.

Sprawdź odpowiedź
1. 32 się mieści w 50 (zostaje 18) -> 1
2. 16 się mieści w 18 (zostaje 2) -> 1
3. 8 się nie mieści -> 0
4. 4 się nie mieści -> 0
5. 2 się mieści (zostaje 0) -> 1
6. 1 się nie mieści -> 0
Wynik: 110010

Moduł 3: Matematyka Binarna

Dodawanie (+)

Zasada: 1 + 1 = 0 (i 1 dalej).

  1 0 1  (5)
+ 0 1 1  (3)
-------
  1 0 0 0 (8)

Odejmowanie (-)

Zasada: 0 - 1 = 1 (pożyczamy od sąsiada z lewej).

📝 Ćwiczenie 3

Oblicz binarnie: 110 + 011 (czyli 6 + 3).

Sprawdź odpowiedź
Od prawej:
0 + 1 = 1
1 + 1 = 0 (1 dalej)
1 + 0 + (1 z pamięci) = 10
Wynik: 1001 (czyli 9)
